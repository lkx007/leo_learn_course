# 项目 05 课：音效 + 游戏结束 🔊

你好，小鱼！👋
游戏能吃了 —— 今天让它 **更像真游戏**：
吃到 **啵** 一声、碰到 **大鲨鱼** 就 **Game Over**！

---

## 🎯 今天目标

1. 吃到鱼 → 播放 **「啵」或「耶」** 音效
2. 加 **Shark** 大鲨鱼（比你还大）—— 碰到就结束
3. 游戏结束 → 说 **「游戏结束，得分：___」** + 停止

---

## 🎤 语音指令（第 05 关）

```text
【角】Scratch 3 老师。
【清】大鱼吃小鱼已有得分和克隆小鱼。
【步】请帮我加：
  1. BigFish 碰到 SmallFish 时，播放声音 Pop（或类似）
  2. 新增 Shark 角色，比 BigFish 大，在屏幕上方左右巡逻
  3. BigFish 碰到 Shark：说「游戏结束！得分：」连得分变量，2秒，停止全部
  4. Shark 用重复执行：x 增加 8，碰到边缘就反弹
【例】像经典大鱼吃小鱼里遇到天敌。
简单就好，我是 7 岁。
```

---

## 🧩 提示

| 功能 | Scratch 积木 |
| :--- | :--- |
| 吃音效 | `播放声音 Pop` 放在「碰到 SmallFish」里 |
| 鲨鱼巡逻 | `重复执行` + `移动` + `碰到边缘就反弹` |
| 游戏结束 | `说` + `停止全部` |
