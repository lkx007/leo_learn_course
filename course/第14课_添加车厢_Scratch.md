# 第14课：添加车厢 (Add/Remove - 增删) ➕➖

## 1. 故事时间 📖
上节课我们搭建了小火车（列表）。
今天我们要学习 **添加车厢** 和 **移除车厢**！

## 2. 列表操作积木（"变量"分类）🚃
| 积木 | 效果 |
|------|------|
| `将 [xxx] 加入 列表` | 加车厢（末尾） |
| `删除 列表 的第 1 项` | 删车厢 |

## 3. 动手搭积木 🧱
```
当绿旗被点击
询问 [要买什么？] 并等待
将 (回答) 加入 购物车
说 连接 [已加入：] 和 (回答) 2 秒
```

## 4. 小挑战 🌟
设计 **"购物车管理"**：
- 输入 `删除` → 删除列表最后一项
- 输入商品名 → 加入购物车

**加油！你就是购物车高手！** 🛒✨
