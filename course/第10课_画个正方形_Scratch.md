# 第10课：画个正方形 (Drawing Shapes - 顺序) 🔲

## 1. 故事时间 📖
今天我们要让小猫 **一边走，一边画画**！

## 2. 添加画笔扩展 🖊️
1. 点左下角 **"添加扩展"**（`+` 号）
2. 选择 **"画笔" (Pen)**
3. 左边会多出一个 **"画笔"** 分类！

## 3. 画笔积木大全 🖊️
| 积木 | 效果 |
|------|------|
| `清空` | 擦掉所有笔画 |
| `落笔` | 开始画 |
| `将画笔颜色设为 [...]` | 换颜色 |

## 4. 动手搭积木 🧱
```
当绿旗被点击
清空
落笔
重复执行 4 次
  移动 100 步
  右转 90 度
抬笔
```

## 5. 小挑战 🌟
设计 **"彩色螺旋"**：`重复执行 100 次` → `移动 (循环计数) 步` → `右转 91 度` → `将画笔颜色增加 10`

**加油！你就是小小画家！** 🎨✨
