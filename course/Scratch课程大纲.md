# Scratch 图形化课程大纲 (6个月 · 24课时) 🐱🧩

## 目标
让小鱼（Leo）在 **完全不打字** 的情况下，先用拖拽积木建立"计算思维"。
**教学理念**：和 Python 主线一一对应——先在 Scratch 里"玩懂"，再到 Python 里"写出"。

> 打开方式：浏览器访问 👉 **[scratch.mit.edu](https://scratch.mit.edu)** → 点"创建"。
> 每节课都有配套 `.sb3` 工程文件（在 `course/` 目录，如 `scratch_lesson_01.sb3`），可在 Scratch 里"作品 → 从电脑中上传"打开参考答案。
