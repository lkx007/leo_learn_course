# 第05课：真假判断练习 🧠🎯

## 判断题

### 第1题
```
10 > 5 是对的吗？
```
**答案：是**

### 第2题
```
8 < 3 是对的吗？
```
**答案：否**

### 第3题
```
7 = 7 是对的吗？
```
**答案：是**

### 第4题
```
5 ≠ 5 是对的吗？
```
**答案：否**
