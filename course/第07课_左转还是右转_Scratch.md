# 第07课：左转还是右转 (If/Else - 否则) 🔄

## 1. 故事时间 📖
上节课学了 **"如果"**（二选一的第一步）。
今天学完整的 **"如果...否则..."** —— 真正的二选一！

## 2. 如果...否则... 怎么用？🧐
```
如果 (条件) 那么
  做了这件事
否则
  做那件事
```

## 3. 动手搭积木 🧱
```
当绿旗被点击
询问 [苹果好吃吗？（好吃/不好吃）] 并等待
如果 (回答) = [好吃] 那么
  说 [我也觉得超好吃！🍎] 2 秒
否则
  说 [啊？你是不是味觉有问题？🤔] 2 秒
```

## 4. 小挑战 🌟
设计 **"智能门卫 2.0"**：
- 密码正确 → 说 `"🎊 欢迎回家！"`
- 密码错误 → 说 `"🚨 警报！警报！非法入侵！"`

**加油！你掌握了二选一的魔法！** 🔄✨
