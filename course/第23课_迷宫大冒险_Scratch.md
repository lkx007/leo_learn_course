# 第23课：迷宫大冒险 (Game - 键盘控制) 🎯🎮

## 1. 故事时间 📖
今天我们要做 **迷宫游戏**！
用 **方向键** 控制小猫，走出迷宫！

## 2. 游戏设计 🎮
| 功能 | 积木 |
|------|------|
| 上键移动 | `当按下 [上箭头] 键` |
| 碰到墙壁？ | `碰到颜色 [...])？` |
| 到达终点？ | `碰到 [终点]？` |

## 3. 动手搭积木 🧱
```
当绿旗被点击
移到 x:-200 y:0
重复执行
  如果 <键 [上箭头] 被按下？> 那么
    将 y 坐标增加 10
  如果 <键 [右箭头] 被按下？> 那么
    将 x 坐标增加 10
```

## 4. 小挑战 🌟
加上 **"碰到墙壁就回到起点"** 的规则！
