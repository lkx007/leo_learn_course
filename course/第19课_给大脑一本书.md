## 📖 什么是 Context (上下文)？

在编程里，我们可以把一本“书”（比如一个很长的文字，或者一个列表）交给 AI 函数。
让它先看书，再回答问题。

## 🧠 带书的 AI

我们给函数加一个新的参数：`book` (书)。

```python
def smart_ai(question, book):
    if question == "我的生日是哪天？":
        # AI 去翻书...
        if "生日" in book:
            print("📖 我在书里找到了！你的生日是 5月1日！")
        else:
            print("🚫 书里没有写呀...")
    else:
        print("🤖 这个问题太难了，书里没有！")
```

## 📝 编写你的“秘密书”

我们可以用一个长长的字符串来当书：

```python
my_diary = "我叫小鱼。我的生日是 5月1日。我最喜欢的颜色是蓝色。我的好朋友是小龟。"

# 召唤 AI，并把日记给它
smart_ai("我的生日是哪天？", my_diary)
```
