你好，小鱼！👋
欢迎来到 **画家海龟** 的世界！🐢

从今天开始，我们要进入一个全新的领域：**图形化编程**。
我们会召唤一只神奇的小海龟，让它在屏幕上画画！

## 🐢 召唤咒语

要召唤小海龟，我们需要两句咒语：

```python
import turtle       # 咒语1：引入海龟魔法包
t = turtle.Turtle() # 咒语2：变出一只海龟，名字叫 t
```

## 🎨 指挥小海龟

小海龟听得懂很多命令，我们先学最简单的三个：

1.  `t.forward(100)`：向前爬 100 步
2.  `t.backward(50)`：向后退 50 步
3.  `t.left(90)`：向左转 90 度（直角）
4.  `t.right(90)`：向右转 90 度

## 🖍️ 画第一笔

让我们试着画一条线，然后转个弯！

```python
import turtle

t = turtle.Turtle()
t.shape("turtle")  # 把箭头变成可爱的小海龟形状
t.color("green")   # 变成绿海龟

t.forward(100)     # 向前爬
t.left(90)         # 左转
t.forward(100)     # 再向前爬

print("海龟已经就位！")
turtle.done()      # 告诉电脑画完了，保持窗口不关闭
```

👉 **注意**：
运行这个程序时，会弹出一个白色的新窗口。这就是海龟的画板！
如果想关闭它，可以点窗口上的小叉叉。
