# 第18课：积木的秘密 (Parameters - 参数) 🔑

## 1. 故事时间 📖
上节课我们做了 `跳一跳` 积木，但它**每次都跳一样高**。
今天我们要给积木加 **"参数"** —— 让积木可以**接收输入**！

## 2. 带参数的自制积木 🔑
制作新积木时，勾选 **"添加输入项"**：
- 输入项名称：`高度`
- 调用时：`跳一跳 (50)` ← 50 就是参数

## 3. 动手搭积木 🧱
```
定义 跳 N 下
  重复执行 (N) 次
    将 y 坐标增加 10
    等待 0.1 秒
```
主程序：`跳 5 下` ← 跳 5 次！

## 4. 小挑战 🌟
制作 `说 N 遍 [消息]` 自制积木，让小猫把一句话说 N 遍！

**加油！你解锁了参数的秘密！** 🔑✨
