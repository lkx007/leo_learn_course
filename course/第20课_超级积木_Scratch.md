# 第20课：超级积木 (Building a Helper - 综合) 🏗️

## 1. 故事时间 📖
今天我们综合运用所有知识，制作一个 **"作业辅导小助手"**！
它会：**聊天 + 算数 + 查资料**！

## 2. 设计思路 🏗️
用 **自制积木** 把三个功能打包：
- `聊天模式`：询问名字，打招呼
- `算数模式`：出加法题，判对错
- `资料模式`：从列表里查关键词

## 3. 动手搭积木 🧱
```
当绿旗被点击
询问 [需要什么帮助？（聊天/算数/资料）] 并等待
如果 (回答) = [聊天] 那么
  聊天模式
否则 如果 (回答) = [算数] 那么
  算数模式
否则
  资料模式
```

## 4. 小挑战 🌟
给小助手加上 **"退出"** 功能：输入 `exit` 就停止程序！

**加油！你就是超级积木大师！** 🏗️✨
