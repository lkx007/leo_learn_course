# 第15课：车厢里有什么？(Indexing - 索引) 🔍

## 1. 故事时间 📖
小火车有好多节车厢，你想知道 **第 1 节** 装了什么吗？
在 Scratch 里，我们用 **"车厢编号"**（索引）来取东西！

⚠️ **注意：Scratch 列表从 1 开始数（不是 0）！**

## 2. 取列表项（"变量"分类）🔍
```
(列表 的第 (1) 项)
```

## 3. 动手搭积木 🧱
```
当绿旗被点击
说 连接 [火车头是：] 和 (我的玩具 的第 (1) 项) 2 秒
说 连接 [火车尾是：] 和 (我的玩具 的第 (列表 的项目数) 项) 2 秒
```

## 4. 小挑战 🌟
让玩家输入一个数字 N，说出 `列表 的第 (N) 项` 是什么！

**加油！你就是检阅大师！** 🔍✨
