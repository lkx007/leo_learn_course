# 第16课：检阅火车 (Iterating - 遍历) 🚂🔍

## 1. 故事时间 📖
如果你想 **把小火车所有车厢都检查一遍**，需要一个一个看吗？
**不用！** 用 **"遍历循环"** 一键检阅！

## 2. 遍历积木（"控制"分类）🔄
```
对于 (i) 在 列表 中 逐一取出
  说 (i) 1 秒
```

## 3. 动手搭积木 🧱
```
当绿旗被点击
说 [开始检阅火车！] 1 秒
对于 (item) 在 我的玩具 中 逐一取出
  说 (item) 1 秒
说 [检阅完毕！] 1 秒
```

## 4. 小挑战 🌟
用遍历循环计算列表里**所有数字的和**！

**加油！你就是遍历大师！** 🔄✨
