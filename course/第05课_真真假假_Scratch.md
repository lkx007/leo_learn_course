# 第05课：真真假假 (True or False - 布尔值) 🧠

## 1. 故事时间 📖
电脑世界里所有判断都是**非黑即白**的：
- 对的 ✅ → **True（真）**
- 错的 ❌ → **False（假）**

## 2. 认识布尔值（绿色"运算"分类）💡
| 比较 | 积木 |
|------|------|
| 等于 | `( 3 ) = ( 5 )` |
| 大于 | `( 10 ) > ( 5 )` |

## 3. 动手试一试 ✍️
```
当绿旗被点击
询问 [你几岁了？] 并等待
如果 (回答) = [7] 那么
  说 [哇，我们同岁！] 2 秒
否则
  说 [你不是 7 岁！] 2 秒
```

## 4. 小挑战 🌟
设计 **"机器人检测仪"**：
- 问：`"你是机器人吗？（是/不是）"`
- 回答 `"是"` → 说：`"嘀嘀嘀！发现机器人！🤖"`
- 回答 `"不是"` → 说：`"欢迎人类朋友！😄"`

**加油！你就是逻辑大师！** 🧠✨
